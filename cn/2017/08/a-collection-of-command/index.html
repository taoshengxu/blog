A collection of Command

2017-08-28

DOS

  • cd / [root]
  • cd ~ [home or root/user]
  • cd . [Current]
  • cd .. [Father]
  • cd - [previous]
  • pwd [current]
  • d: +Enter [D:/]

Ubuntu

Basic

  • sudo su 进入root
  • exit or logout or ctrl D 退出root

Download

  • & 加在一个命令之后可以把这个命令放到后台运行
  • Ctrl+Z 可以把一个前台执行的命令放到后台
  • jobs -l 查看后台运行的命令
  • fg %jobnumber 将后台命令调到前台
  • bg %jobnumber 重启后台暂停的命令
  • kill %num 杀死
  • ps 查看进程号PID
  • nohup 始终执行
  • ps - aux | grep xxx 显示终端关闭后的后台进程

磁盘管理

  • mkdir -p 递归创建目录
  • cp 复制
  • rmdir 删除空目录
  • df -lh 查看磁盘空间

  • sudo fdisk -lu 显示硬盘及所属分区情况

  • ubuntu 加载新硬盘教程Here and There

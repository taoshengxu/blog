Git 参考手册中文
Git 参考手册ENG

  • git check out -b 创建新分支，并立即切换到它。与以下等效：
    • git branch newbranch
    • git checkout newbranch
  • git merge [branch] 将[branch] 分枝合并到当前分支中
  • git push [alias] [branch]，就会将你的[branch]分支推送成为[alias]远端上的[branch] 分支
